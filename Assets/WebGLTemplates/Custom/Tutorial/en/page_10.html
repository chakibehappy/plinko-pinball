Plinko Pinball

Game Controls

Tap the +/- buttons to increase or decrease the bet amount.
Tap to start the game round. Each game round consists of 5 balls.
Tap to enter autoplay selection menu.
Select the number of game rounds, then tap Confirm to start autoplay. The ball will launch automatically in autoplay mode.
Press or hold the button, then release to launch the ball.
Display the number of balls remaining in the game round. Tap to expand the menu and view the bet value for each ball along with the account balance.
