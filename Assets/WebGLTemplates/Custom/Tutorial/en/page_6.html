Plinko Pinball

Game Elements

Whichever of the four key elements—Bumpers, Bulls Eye, Pin, or Alien—is hit first will upgrade the second-highest yellow-framed multiplier in the current prize table:

  • Set 1: 2.5x upgrades to 5.0x
  • Set 2: 2.5x upgrades to 5.0x
  • Set 3: 1.2x upgrades to 2.2x

If another ball hit the elements, the mega bonus is unlocked, upgrading the highest yellow-framed multiplier in the Multiplier Prize Table to a higher value:

  • Set 1: 4x upgrades to 20x
  • Set 2: 10x upgrades to 50x
  • Set 3: 10x upgrades to 170x
