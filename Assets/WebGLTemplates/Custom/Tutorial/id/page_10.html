Plinko Pinball

Kontrol Game

Ketuk tombol +/- untuk menambah atau mengurangi jumlah taruhan.
Ketuk untuk memulai putaran permainan. Setiap putaran terdiri dari 5 bola.
Ketuk untuk membuka menu pemilihan putar otomatis.
Pilih jumlah putaran permainan, lalu ketuk Konfirmasi untuk mulai putar otomatis. Bola akan meluncur otomatis dalam mode ini.
Tekan atau tahan tombol, lalu lepaskan untuk meluncurkan bola.
Menampilkan jumlah bola tersisa dalam putaran permainan. Ketuk untuk memperluas menu dan lihat nilai taruhan per bola serta saldo akun.
