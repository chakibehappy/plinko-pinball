Elemen Permainan

Elemen kunci manapun dari empat berikut — Bumpers, Bulls Eye, Pin, atau Alien — yang terkena pertama kali akan meningkatkan pengganda kedua tertinggi berbingkai kuning pada tabel hadiah saat ini:

  • Set 1: 2.5x meningkat menjadi 5.0x
  • Set 2: 2.5x meningkat menjadi 5.0x
  • Set 3: 1.2x meningkat menjadi 2.2x

Jika bola lain mengenai elemen-elemen tersebut, mega bonus akan terbuka, meningkatkan pengganda tertinggi berbingkai kuning pada Tabel Hadiah Pengganda ke nilai yang lebih tinggi:

  • Set 1: 4x meningkat menjadi 20x
  • Set 2: 10x meningkat menjadi 50x
  • Set 3: 10x meningkat menjadi 170x
